Selected Data for Eleven Wildcat Wells in the National Petroleum Reserve in Alaska

USGS Open File Report 99-015

Disc Information

This CD-ROM makes available selected data sets from eleven wildcat wells in the National Petroleum Reserve in Alaska (NPRA). The CD-ROM is designed strictly as a "data archive" with a simple HTML interface to aid in navigating the disc.

The CD-ROM was written to the ISO 9660 standard and was produced to work in the greatest number of environments. The HTML was written as generically as possible in order to work with as many browsers and operating systems as possible. The Adobe Acrobat PDF files were kept below 4 megabytes (MB's) in order to avoid the problem some browsers have with PDF files greater than that size. Further, the smaller file sizes facilitate serving this disc over the Internet.

The following data types are provided for each well:

  • Photographs of Cores
  • Depths to Selected Stratigraphic Horizons
  • Geological and Drilling Reports
  • LAS-Format Core-Gamma Ray Data
  • LAS-Format Borehole-log Data
  • Permeability, Porosity, and Grain Density Data

All files are ASCII text with the exception of the Core Image files, which are .GIF binary image file format, and the Drilling History and Geological Reports, which are Adobe Acrobat PDF files.

For best viewing, all ASCII files should be viewed using a fixed size font, such as Courier.

Disc Structure

The files on this disc are arranged using the following folder hierarchy:

  • Root (Top Level) Folder
  • Docs Folder
  • Wells Folder
  • {Well Folder Name} Folder
  • Core Folder
  • GammaRay Folder
  • LAS Folder
  • PDF Folder
  • PNP Folder
  • Tops Folder

    Folder Descriptions

    Root (Top Level) Folder
    The Root, or Top level, Folder contains:
    • Docs subfolder
    • Wells subfolder
    • WorkShop.htm - the "homepage" for this disc.
    • README.txt - an introduction to this disc.

    Docs Folder
    The Docs folder contains HTML files which provide:
    • Documentation about data sources.
    • Acnowledgements.
    • Disclaimer.
    • Author Information.

    Wells Folder
    The Wells folder contains:
    • WellIdx.htm: an HTML index to the well folders.
    • A subfolder for each well included on this disc.
      Most files contained in a well subfolder and its subfolders are named starting with a prefix indicating the well to which the file applies. Well folder names and prefixes for the individual wells included on this disc are:

    Each of the Well Folders listed in the table above contains:
    • README.txt
    • Index.htm - An HTML index of the individual subfolders shown below.
    • {PREFIX}Doc.htm - An HTML file documenting the data sources for the data in the individual subfolders shown below.

      Graphic image files of well core.
      • GIF-format graphic image files of well cores photographs.
      • {PREFIX}Core.htm - An HTML file which indexes the core image files.
      • {PREFIX}Core.txt - A text file which indexes the core image files.
      • README.txt
    • GammaRay Folder
      Well core gamma ray data.
      • {PREFIX}GR.htm - An HTML file which describes the well core gamma ray data.
      • {PREFIX}GR.las - A LAS-format log file of the well core gamma ray data.
      • README.txt
    • LAS Folder
      LAS-format well log data.
      • {PREFIX}LAS.htm - An HTML file which describes the well log data.
      • {PREFIX}.las - A LAS-format log file of the well log data.
      • README.txt
    • PDF Folder
      Geological and Drilling History reports in Adobe Acrobat PDF format.
      • {PREFIX}DRILL.pdf - Drilling history.
      • {PREFIX}GEOL.pdf - Geological report.
      • {PREFIX}PDF.htm - An HTML file which indexes the PDF files.
      • README.txt
    • PNP Folder
      Permeability and Porosity Data
      • {PREFIX}PNP.htm - An HTML file which describes the permeability and porosity data.
      • {PREFIX}PNP.txt - A text file which describes the permeability and porosity data.
      • README.txt
    • Tops Folder
      Depths to Selected Stratigraphic Horizons
      • {PREFIX}Tops.htm - An HTML file which describes the stratigraphic horizons data.
      • {PREFIX}Tops.txt - A text file which describes the stratigraphic horizons data.
      • README.txt
